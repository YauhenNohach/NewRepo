Животные:
Животное содержится в одном вольере.
Животные могут быть привязаны к зоологу, который за ними ухаживает.

Вольеры:
В одном вольере могут содержаться несколько животных.

Зоологи:
Зоолог ухаживает за несколькими животными.
У каждого зоолога есть имя, специализация и стаж работы.

Зоны зоопарка:
Каждая зона включает несколько вольеров.

Посещения:
Каждый билет позволяет посетить несколько зон зоопарка.
