Команды:

Каждая команда участвует в матчах.
У каждой команды тренер и состав игроков.

Игроки:

Каждый игрок может быть только в одной команде.

Матчи:

Матч проходит между двумя командами и имеет дату и счёт.
Система должна учитывать, на каком стадионе прошёл матч.
Список всех игроков, участвовавших в матче

Сезоны:

Матчи привязаны к сезону.
