Экспонаты (Exhibit):
Экспонат может быть частью одной или нескольких выставок.

Выставки (Exhibition):
Выставка организуется определённым куратором.
Выставка может включать несколько экспонатов.

Кураторы (Curator):
Куратор отвечает за одну или несколько выставок.

Залы музея:
У музея есть залы, где размещаются выставки.
Каждый зал может быть задействован для одной или нескольких выставок одновременно.

Посетители:
Посетители могут посещать одну или несколько выставок.
